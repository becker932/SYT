WP Symposium Pro Sidebar Widget for user info display

== Description ==

The WP Symposium Pro Profile Widget is used in conjunction WP Symposium Pro Plugin. It allows you to display user information and links and by adding additional shortcodes you can display just about anything in a themes sidebar.

	1. Displays user Avatar.
	2. Alows you to choose menu to display below Avatar.
	3. Displays friends list up to 5.
	4. You have 6 (six) additional Slots to enter shortcodes with a heading slot above each.

A few notes about the sections above:

== Installation ==

Go to plugins in your admin dashboard and click Add New. Choose upload and select the wp-symposium-pro-profile-widget.zip file.
